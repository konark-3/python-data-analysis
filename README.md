# Data Analysis with Python - Weather & Squirrel Census

A Python project demonstrating **data analysis and manipulation** using CSV files and Pandas. The project covers extracting insights from weather data and wildlife census data.

---

## Tech Stack

- **Language:** Python 3.x  
- **Libraries:** `pandas`, `csv`  

---

## Key Highlights

- Extract and analyze temperature data from `weather_data.csv`.  
- Convert Celsius to Fahrenheit and find peak temperatures.  
- Analyze squirrel census data by primary fur color.  
- Generate summary CSV reports (`NewData.csv`).  

---

## How to Run

1. Ensure CSV files are in the project directory:
   - `weather_data.csv`  
   - `2018_Central_Park_Squirrel_Census_-_Squirrel_Data_20240620.csv`  
2. Install dependencies:


pip install pandas

## Run the Script


python data_analysis.py

## Outcome

- Lists of temperatures and conversions  
- Day with maximum temperature highlighted  
- Summary CSV file with squirrel counts by fur color
